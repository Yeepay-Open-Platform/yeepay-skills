# 快速接入

本文是易宝开放平台接入的**流程导航**。各步骤细节见对应专题文档，避免在本页重复展开。

## 流程概览

```mermaid
graph LR
    A[商户入网并签约产品] --> B[密钥配置]
    B --> C[SDK集成]
    C --> D[开发对接]
    D --> E[测试验证]
    E --> F[正式上线]
```

## 一、商户入网并签约产品

1. 登录 [控制台](https://open.yeepay.com/developer-center) → **开发服务** → **接入诊断**，确认产品与权限已开通（详见 `工具与支持/开发工具/接入诊断.md`）。
2. 在 **我的应用** 确认 AppKey 与应用权限（详见 `应用管理.md`）。

## 二、密钥配置

1. **选型与制备**：RSA 或 SM 见 `密钥管理/密钥介绍.md`；工具操作见 `工具与支持/开发工具/密钥工具.md`。
2. **控制台配置**：RSA → `密钥管理/密钥配置(RSA).md`；SM → `密钥管理/密钥配置(SM).md`。
3. **安全提示**：私钥本地保管，勿泄露、勿写入代码仓库。

## 三、SDK 集成

| 场景 | 文档 |
| --- | --- |
| Java | `开始对接/SDK使用说明.md` |
| 其他语言 | `工具与支持/开发工具/平台SDK.md` |
| 无官方 SDK / 自研网关 | `平台规范/安全认证/` 下的请求签名协议/回调解密协议 |

可选：VSCode YOP 插件辅助生成示例代码（[插件说明](https://open.yeepay.com/docs/platform/developTools/developer_tools_vscode)）。

## 四、开发对接

1. **接口字段与业务规则**：以在线 `doc_md` 为准（技能内通过 `api-index.yaml` 定位）。
2. **出站签名/加密**：RSA → `平台规范/安全认证/鉴权认证机制(RSA).md`；SM → `鉴权认证机制(SM).md`。
3. **异步通知**：机制见 `平台规范/结果通知机制说明.md`；协议见 `结果通知(RSA).md` 或 `结果通知(SM2).md`；入站网络见 `平台规范/回调网络配置.md`。
4. **出站 IP 白名单**（资金类接口）：`开始对接/配置IP白名单.md`。

## 五、测试验证

- **沙箱联调**：`开始对接/沙箱环境联调测试.md`
- **平台错误码**：`开始对接/平台错误码说明.md`
- **SDK 排障**：`开始对接/Java-SDK报错说明.md`
- **通知查询/重发**：`平台规范/结果通知查询与重发.md`

## 六、正式上线

上线前检查见 `平台规范/上线前检查清单.md`；联调阶段可先对照 `../../troubleshooting.md` §七 关键项自检。
