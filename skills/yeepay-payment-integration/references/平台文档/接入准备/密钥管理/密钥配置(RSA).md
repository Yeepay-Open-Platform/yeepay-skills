# 配置商户密钥（RSA）

控制台入口：[控制台](https://open.yeepay.com/developer-center) → **我的应用** → **密钥配置**。

## 一、准备工作

1. 登录商户后台所需的账号信息。
2. 已制备商户 RSA 密钥或 CFCA 证书：
   - 普通密钥：`密钥介绍.md` §四 + `密钥工具.md`，或联调 `scripts/rsa/gen_keypair.py`。
   - 高风险接口（代付代发等）：须 CFCA 证书，见 `CFCA证书制备.md`。

## 二、首次配置

**路径**：登录 [控制台](https://open.yeepay.com/developer-center) → **我的应用** → **密钥配置**。

1. 选择 **密钥类型** 为「国际密钥（RSA）」。
2. **密钥格式** 二选一：
   - **文本**：在 **公钥内容** 文本框粘贴 RSA 商户公钥（PKCS#8 / X.509 Base64）。
   - **证书**：**密钥格式** 选「证书」，在 **公钥证书** 下拉按序列号选择 CFCA 证书；列表为空请先完成 `CFCA证书制备.md`。
3. 点击 **保存**，完成 **绑定手机号** 短信验证。
4. 页面提示配置成功。

## 三、修改商户密钥

私钥泄漏等情况下在 **密钥配置** 页点击 **设置公钥**：

1. 录入新公钥（文本或证书，步骤同首次配置）。
2. 完成手机号验证。
3. 阅读变更说明并评估对回调的影响（见下「注意事项」）。
4. 确认后完成修改。

**变更顺序建议**：

- 对回调不敏感：可先改密钥，再上线新回调代码（平台 24 小时内有失败重试）。
- 对回调敏感：须先上线支持新密钥的回调代码，再修改商户密钥。

## 四、立即吊销 / 撤销变更

- **立即吊销**：修改成功后弹窗点 **立即吊销** → 手机号验证 → 二次确认。旧密钥类型为「文本」则吊销旧商户密钥；为「证书」则同时吊销旧密钥与旧 CFCA 证书。
- **撤销变更**：若新密钥导致回调异常，点 **撤销变更** 回退到旧密钥。

## 五、验证公钥与摘要信息

- **验证公钥**：密钥配置页点 **验证公钥**，用商户私钥对字符串 `a=123` 生成签名并粘贴，点 **验证** 确认公私钥匹配。
- **摘要信息**：点 **摘要信息** 查看商户公钥指纹，核对是否上传正确公钥。

## 六、查看易宝公钥

密钥类型为 RSA 时，可 **查看** 并 **复制** 易宝平台公钥，用于不使用 SDK 时的回调验签或手工加密。使用 Java SDK 最新版时通常由 SDK 自动管理，无需本地配置。
