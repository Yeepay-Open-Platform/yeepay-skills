# 配置商户密钥（SM）

控制台入口：[控制台](https://open.yeepay.com/developer-center) → **我的应用** → **密钥配置**。

## 一、准备工作

1. 登录商户后台所需的账号信息。
2. 已制备 **SM2 格式 CFCA 证书**（见 `CFCA证书制备.md`）。

## 二、首次配置

**路径**：登录 [控制台](https://open.yeepay.com/developer-center) → **我的应用** → **密钥配置**。

1. **密钥类型** 选择「国密（SM2）」。
2. **密钥格式** 选择「证书」，在 **公钥证书** 下拉按序列号选择已激活的 CFCA 证书；列表为空请先完成 `CFCA证书制备.md`。
3. 点击 **保存**，完成 **绑定手机号** 验证。
4. 页面提示配置成功。

## 三、修改商户证书

证书临近过期或私钥泄漏时，在 **密钥配置** 点击 **设置公钥**：

1. **密钥格式** 选「证书」，选择新 **公钥证书**，点 **保存**。
2. 完成手机号验证。
3. 阅读变更说明（评估回调敏感度，同 RSA 文档 §三）。
4. 确认后完成修改。

## 四、立即吊销 / 撤销变更

- **立即吊销**：修改成功后弹窗 **立即吊销** → 验证手机 → 二次确认，吊销旧商户密钥。
- **撤销变更**：回调异常时可 **撤销变更** 回退到旧证书。

## 五、验证公钥与摘要信息

- **验证公钥**：用商户私钥对 `a=123` 签名，在控制台 **验证公钥** 粘贴签名验证匹配。
- **摘要信息**：查看商户公钥摘要，核对证书是否正确。
