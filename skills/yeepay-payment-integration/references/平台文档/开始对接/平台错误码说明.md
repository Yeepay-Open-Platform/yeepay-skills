# 平台公共错误码

> 注意：未使用平台提供官版 SDK 的开发者，请在请求头中添加 `x-yop-sdk-langs=<开发语言>`（比如 go、python、php 等）和 `x-yop-sdk-version=3.0.0`，YOP 网关将返回含义更加明确的【新版】错误码。
