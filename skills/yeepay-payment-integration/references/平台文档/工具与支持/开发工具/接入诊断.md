# 接入诊断

## 概述

接入诊断用于检查应用配置、产品与解决方案 API 开通情况、单个 API 开通状态，并给出解决方案指引。

## 进入页面

1. 登录 [控制台](https://open.yeepay.com/developer-center)。
2. 菜单路径：**开发服务** > **接入诊断**。

## 应用诊断

1. 在接入诊断页顶部的**应用**下拉框中选择目标应用。
2. 点击 **执行诊断**。
3. 查看诊断结果列表；若某项异常，按结果中的**解决方案**链接或说明操作。

## API 诊断

### 方式一：按「产品或方案」诊断

1. 选择目标**应用**。
2. **诊断类型**选择 **产品或方案**。
3. 输入要接入的产品或方案名称，点击 **执行诊断**。
4. 切换到 **API** 标签，查看该产品/方案下各 API 的开通状态。

### 方式二：按「API」诊断

1. 选择目标**应用**。
2. **诊断类型**选择 **API**。
3. 输入 API 路径/名称；或在开放平台 **API 详情页**右上角点击 **接入诊断** 跳转带入。
4. 点击 **执行诊断**。
5. 在 **API** 标签查看该 API 在当前应用下的开通情况。

## 常见结论

- **未开通产品/接口**：在控制台申请对应产品或联系商务/技术支持开通。
- **密钥未配置**：按 `接入准备/密钥管理/` 章节完成 RSA 或 SM 密钥配置。
- **IP 白名单拦截**：资金类敏感接口需配置出口 IP，见 `开始对接/配置IP白名单.md`。
