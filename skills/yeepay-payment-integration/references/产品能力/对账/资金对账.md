# 资金对账

申请并下载资金账单（先申请、后下载，两步异步）。

> 接口字段以在线文档为准：按下表 catalog id 在 `../api-index.yaml` 取其 `doc_md` 链接，执行
> `curl -sS "<doc_md>"` 后再实现（单文件含字段/示例/错误码/示例代码）。

## 场景 → 接口

| 用途 | catalog id | 方法 | 路径 |
|------|-----------|------|------|
| 资金账单申请 | `bill-fund-apply` | POST | `/rest/v1.0/std/bill/fundbill/apply` |
| 资金账单下载 | `bill-fund-download` | GET | `/yos/v1.0/std/bill/fundbill/download` |

## 流程

1. 调 `bill-fund-apply` 申请资金账单（异步生成）。
2. 等待生成完成后调 `bill-fund-download` 下载。
3. 以易宝资金账单为基准，与本地资金流水勾兑。

## 必读规则

1. `../../平台文档/平台规范/安全认证/鉴权认证机制(RSA).md`（请求签名）。
2. `../../平台文档/工具与支持/最佳实践/文件下载.md`（文件下载处理）。

## 易错点

- 资金账单为「先申请后下载」两步，注意异步生成时延，未生成完成时下载会失败。
- 申请域为 `openapi`、下载域为 `yos`，网关不同（见 catalog `gateway`）。
- 字段口径、金额单位与符号以在线文档为准。
